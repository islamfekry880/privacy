Captain Pro – نسر الطريق ("the App") is developed by FananSoft. This Privacy Policy explains what information is collected, how it is used, and how we protect your data when you use the application.

1. Information We Collect

The application may collect the following information:

  • Phone number used to create your account.
  • Unique device identifier for license verification.
  • Device model.
  • Android version.
  • Application version.
  • Subscription and trial information.
  • Anonymous application usage events required for improving stability.

2. Accessibility Service

Captain Pro uses Android Accessibility Service exclusively for detecting ride offer screens and assisting with automatic ride analysis.

The Accessibility Service is NOT used to:

  • Read personal conversations.
  • Collect passwords.
  • Access banking information.
  • Monitor user activity outside supported ride applications.

3. Screen Capture Permission

The application requests Screen Capture permission only to analyze ride offers displayed by supported ride-hailing applications.

Captain Pro does NOT:

  • Record screen videos.
  • Store screenshots.
  • Upload screen recordings.

Only the minimum information required to calculate trip profitability is processed.

4. Why We Collect Information

  • User authentication.
  • Subscription verification.
  • License validation.
  • Preventing unauthorized access.
  • Improving application performance.
  • Providing ride analysis.

5. Data Security

User information is transmitted securely using encrypted HTTPS connections. Sensitive account information is protected using industry-standard security practices.

6. Third-Party Services

Captain Pro uses secure backend infrastructure provided by Supabase for authentication, subscription verification, and cloud services.

7. Data Sharing

We do not sell, rent, or trade your personal information. Information is only processed when required to provide the application's functionality.

8. Data Retention

User information is retained only for as long as necessary to provide the service, comply with legal obligations, resolve disputes, and enforce our agreements.
